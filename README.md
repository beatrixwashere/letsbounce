# letsbounce

four key rhythm game with notes that can bounce for extra hits, including bounces into separate lanes. still very wip and i'll be workin on this some more but i dont plan on continuing it on my own so issues and prs are appreciated :3

(for an example map: download `boyletmeknow.lbm` from the root folder)

**controls:**
- dfjk // gameplay keys
- up/down or scroll wheel // editor movement
- shift // used with above; 5 steps instead of 1
